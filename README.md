## Animal Sounds
内容はとってもシンプル。
動物の音が流れるので、その音に合う動物さんを選択肢の中から選ぶゲームです。

## URL
実際のゲームはこちらのサーバーにアップしています。
もしご興味あれば遊んでみてください。

https://animal-sounds.cocoroworks.net/

# 動作環境について
node v14.0.0以上推奨

### `npm start`
サーバー起動

### `npm run build`
buildファイルを生成
