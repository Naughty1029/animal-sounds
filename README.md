## Animal Sounds
内容はとってもシンプル。
動物の音が流れるので、その音に合う動物さんを選択肢の中から選ぶゲームです。

## URL
実際のゲームはこちらのサーバーにアップしています。
もしご興味あれば遊んでみてください。

https://animal-sounds.cocoroworks.net/

# 使用技術について
React　v17.0.2
TypeScript　v4.5.5
Material UI v5.4.2
Emotion 11.8.1
ESLint/Prettier

### `npm start`
サーバー起動

### `npm run build`
buildファイルを生成
